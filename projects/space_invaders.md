# Alien Invaders (Space Invaders Clone) 
<hr>
## Languages & Tools Used

**Languages Used:** C++  
**Libraries Used:** SFML, Chronos  
**Developed On:** Linux  
[Github Page](https://github.com/charles-mowbray/alien-invaders-final)  
## Overview
- CSCI 261 Final Project  
- Rebuilt Space Invaders in C++ using just the SFML Window Library  
- Added Unique timing and some slight modifications to the players rate of fire and enemy rate of fire  


## Description
Rebuilt the classic arcade game Space Invaders in C++ as a final project for my freshman year programming course. 
Where I got to pracitice the fundamentals of object oriented programming.
